## Run application
```sh
$ yarn dev
```

## Run test
```sh
$ yarn test
```

## Run lint
```sh
$ yarn lint
```

## Run lint:fix
```sh
$ yarn lint:fix
```

## Run migration
```sh
$ yarn migration
```

## Run migration:revert
```sh
$ yarn migration:revert
```

## Run create migration
```sh
$ typeorm migration:create -n NameMigration
```
